# 🎫 HelpDesk Ticketing System

A simple and efficient **IT Support Ticketing System** built using the **MERN Stack (MongoDB, Express, React, Node.js)**.

This project allows users to raise issues, and support teams to manage and resolve them efficiently.

## 🚀 Key Features

* 🔐 Secure Login & Registration (JWT Authentication)
* 👤 Role-based access:

  * Admin
  * Support Agent
  * User
* 🎫 Create and manage support tickets
* 📌 Track ticket status (Open → In Progress → Resolved)
* 💬 Add comments to tickets
* 📱 Fully responsive design (works on mobile & desktop)

---

## 👥 User Roles

### 👤 User

* Create tickets
* View own tickets
* Add comments

### 🧑‍💻 Support Agent

* View assigned tickets
* Update ticket status
* Respond to users

### 👑 Admin

* Full control over all tickets
* Assign tickets to agents
* Manage system

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Authentication:** JWT
* **Deployment:**

  * Frontend → Netlify
  * Backend → Render

## ⚙️ How to Run Locally

### 1. Clone the project

git clone https://github.com/sahazadalam/helpdesk-ticketing-system.git
cd helpdesk-ticketing-system

---

### 2. Backend setup

cd backend
npm install

Create `.env` file:

MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret

Run server:

npm start

---

### 3. Frontend setup

cd frontend
npm install
npm start
